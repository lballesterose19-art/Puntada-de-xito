# Puntada de Exito

Starter inicial para desarrollar la web publica y el prototipo de app del proyecto **Puntada de Exito**, una marca de productos textiles premium y servicios de costura.

## Estructura

- `web/`: primera version de la web publica.
- `app-prototype/`: prototipo navegable de la app en HTML, CSS y JavaScript.
- `assets/`: imagenes y recursos visuales del proyecto.
- `docs/`: notas de producto y siguientes decisiones.

## Como verlo

Abre estos archivos en el navegador:

- `web/index.html`
- `app-prototype/index.html`

No requiere instalar dependencias todavia.

Si el servidor local esta activo, tambien puedes ver:

- `http://localhost:4173/web/`
- `http://localhost:4173/app-prototype/`

## Siguientes pasos recomendados

1. Definir si la app se construira como React Native, Flutter o PWA.
2. Convertir este prototipo a componentes reales.
3. Definir funcionalidades principales: catalogo, cotizaciones, carrito, pedidos, pagos, servicios y perfiles.
4. Preparar identidad visual: logo, tipografias, paleta y tono de comunicacion.

## Para Lanzamiento

Revisa `docs/lanzamiento.md` antes de publicar. Ahi estan los datos reales que faltan, el stack recomendado y el checklist para salir al mercado.
